# lidar-test-webserial

LiDARユニットGS2を使用したWeb Serial APIのテストプロジェクトです。

## デモ
https://code4fukui.github.io/lidar-test-webserial/

## 機能
- LiDARユニットGS2からのデータ取得
- 取得データの可視化

## 必要環境
- Web Serial API対応ブラウザ

## 使い方
1. ブラウザでプロジェクトページを開く
2. `requestPort`ボタンをクリックし、接続するシリアルポートを選択
3. `startScan`ボタンをクリックしてスキャンを開始
4. 取得データが可視化される
